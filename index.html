HTML QUESTIONS

What is HTML?
  • HTML stands for HyperText Markup Language. It's the standard language used to create web pages. It defines the structure and content of a webpage, including text, images, links, and other elements.
Why do we use HTML?
  • HTML is essential for building web pages because it provides the framework for organizing and displaying content. It allows you to structure text, add images and media, create links, and define the layout of your webpage.
What is doctype in HTML?
  • The doctype declaration specifies the document type definition (DTD) of an HTML document. It tells the web browser which version of HTML is being used. For example, <!DOCTYPE html> indicates that the document is using HTML5.
What is the root element of HTML?
  • The root element of HTML is <html>. It encompasses all other elements in an HTML document.
What is a tag?
  • A tag is a keyword surrounded by angle brackets (< >) that defines an element in HTML. It specifies the start and end of an element. For example, <p> and </p> are tags that define a paragraph.
Give me a few examples of self-closing tags?
  • Self-closing tags are tags that don't require a closing tag. They are often used for elements that don't contain content, such as images and line breaks. Here are a few examples: <img> (image)
    <br> (line break)
    <hr> (horizontal rule)
    <input> (input field)
    <meta> (metadata)
What is an element?
  • An element is a unit of content in an HTML document, defined by a pair of opening and closing tags. Elements can contain text, other elements, or attributes.
    For example, <p>This is a paragraph.</p> is an element.
How many types of elements are there in HTML?
  • There are many types of elements in HTML, each serving a specific purpose. Some common types include: Text elements: <p>, <span>, <h1>, <h2>, etc.
    Block elements: <div>, <header>, <footer>, <section>, etc.
    Inline elements: <a>, <img>, <span>, <strong>, etc.
    Form elements: <form>, <input>, <label>, <button>, etc.
    Semantic elements: <article>, <aside>, <nav>, <main>, etc.
What is the difference between inline and block elements?
  • Inline elements: These elements are displayed in a single line and do not start on a new line. They are typically used for small pieces of content that fit within a line of text.

  • Block elements: These elements start on a new line and occupy the full width of their container. They are used for larger sections of content that should be displayed separately.
What is the difference between semantic and non-semantic elements?
  • Semantic elements: These elements convey meaning about their content. For example, <article> indicates an article, <nav> indicates navigation, and <header> indicates a header.

  • Non-semantic elements: These elements do not have any inherent meaning. They are used primarily for styling and layout purposes. For example, <div> and <span> are non-semantic elements.
What are href vs src attributes?
  • href (Hypertext Reference): This attribute is used to specify the target URL for a hyperlink. It's commonly used with the <a> tag to create links to other web pages, files, or resources.
  • src (Source): This attribute is used to specify the URL of a resource, such as an image, audio file, or video. It's commonly used with the <img>, <audio>, and <video> tags.
What are the features of HTML 5?
    HTML 5 introduced several significant features and enhancements, including:
  • New semantic elements: This includes elements like <header>, <footer>, <nav>, <article>, <section>, and <aside>, which provide better structure and meaning to web content.
  • Video and audio support: HTML 5 provides native support for embedding video and audio content directly into web pages.
  • Canvas element: This allows for dynamic, client-side rendering of graphics.
  • Web Storage API: This enables local storage of data on the client's device, providing offline functionality.
  • Geolocation API: This allows web applications to access the user's geographical location.
  • Web Workers API: This enables JavaScript code to run in background threads, improving performance.
  • Drag and drop API: This allows users to drag and drop content within web pages.
  • Server-Sent Events: This allows web servers to push updates to clients in real-time.
What are entities in HTML?
  • Entities are special characters used in HTML to represent characters that are not easily represented using plain text. They are typically used to represent characters that have special meanings in HTML, such as quotation marks, ampersands, and angle brackets. For example, & lt; represents the less than symbol, and & gt; represents the greater than symbol.
What is div vs span?
  • div (Division): This is a block-level element that is used to group other elements together. It is often used for layout purposes.
  • span: This is an inline element that is used to group other elements together within a line of text. It is often used for styling or formatting purposes.
What are list tags in HTML?
    List tags are used to create lists of items in HTML. There are two main types of lists:
  • Ordered lists: These lists are numbered or lettered. The <ol> tag is used to create ordered lists.
  • Unordered lists: These lists are marked with bullet points. The <ul> tag is used to create unordered lists.
What are attributes?
  • Attributes are additional properties that can be added to HTML elements to modify their behavior or appearance. They are specified within the opening tag of an element, separated by spaces. For example, the <img> tag can have attributes like src, alt, width, and height.
Mention a few attributes for img tags?
  • src: Specifies the URL of the image.
    alt: Provides an alternative text description for the image, which is important for accessibility.
    width: Sets the width of the image in pixels.
    height: Sets the height of the image in pixels.
    title: Provides a tooltip that appears when the user hovers over the image.
What is thead on a table?
  • The <thead> element is used to define the header row or rows of a table. It typically contains the table's column headers.
What is tr?
  • The <tr> element is used to define a table row. Each row contains one or more table cells.
What is th?
  • The <th> element is used to define a table header cell. Header cells are typically displayed in bold or a different font style.
What is td?
  • The <td> element is used to define a table data cell. Data cells contain the actual content of the table.
Can I use th in tbody?
  • No, you cannot use <th> elements within <tbody>. <th> elements are specifically designed for table headers, and they should be placed within the <thead> element. Using them within <tbody> would be incorrect and could lead to unexpected results.
What are meta tags?
    Meta tags are HTML tags that provide metadata about an HTML document. They are placed within the <head> section of an HTML document and are used to provide information to search engines, browsers, and other applications. Some common meta tags include:
  • <meta name="description">: Provides a brief summary of the webpage's content.
    <meta name="keywords">: Specifies relevant keywords related to the webpage's content.
    <meta name="viewport">: Controls the viewport settings for mobile devices.
    <meta charset="UTF-8">: Specifies the character encoding used in the document.
What is SEO?

It involves various techniques, such as keyword research, on-page optimization, link building, and technical SEO, to make a website more attractive to search engines and users.

What is an iframe and anchor?
  • Iframe (Inline Frame): An iframe is an HTML element that allows you to embed a webpage within another webpage. It creates a separate browsing context within the main page, allowing you to display content from different sources.
    Anchor: An anchor is a link within a webpage that, when clicked, takes the user to a specific location within the same page or to a different page altogether. It is typically created using the <a> tag with the href attribute specifying the target URL.
What is Git vs GitHub?
  • Git: Git is a version control system that tracks changes to files over time. It allows you to manage your code projects efficiently, collaborate with others, and revert to previous versions if needed.
  • GitHub: GitHub is a web-based platform that provides hosting for Git repositories. It offers additional features like issue tracking, project management, and collaboration tools, making it a popular choice for developers.
What is the workflow of GitHub?
    The typical GitHub workflow involves the following steps:
  1. Create a repository: Create a new repository on GitHub to store your project's files.
  2. Clone the repository: Clone the repository to your local machine using the Git clone command.
  3. Make changes: Make changes to your project files.
  4. Stage changes: Use the git add command to stage the changes you want to commit.
  5. Commit changes: Use the git commit command to create a snapshot of your staged changes.
  6. Push changes: Use the git push command to upload your committed changes to the remote GitHub repository.
What command do we need to use to move the changes to staged changes?
  • To move changes to staged changes, you use the git add command. For example, to stage all changes in the current directory, you would use: git add .
How can you stage all files at a time?
  • To stage all files at a time, you can use the git add . command, as mentioned above. This will add all modified files in the current directory and its subdirectories to the staging area.
What is the git push command and its usage?
  • The git push command is used to upload your committed changes to a remote repository, such as GitHub. It synchronizes your local repository with the remote repository, making your changes accessible to others. The basic usage is: git push origin main
What is git commit command usage?
  • The git commit command is used to create a snapshot of your staged changes and save them to your local repository. It requires a commit message to describe the changes. The basic usage is: git commit -m "Your commit message"
    Replace "Your commit message" with a meaningful message that describes the changes you made.
What is the use of the git init command?
  • The git init command is used to initialize a new Git repository in a directory. This creates a .git subdirectory within the directory, which stores the version control information for your project. Once you've initialized a repository, you can start tracking changes to your files using Git commands.
What are formatting tags?
    Formatting tags are HTML tags used to apply specific styles or formatting to text content. Some common formatting tags include:
  • <b>: Bold text
  • <i>: Italic text
  • <u>: Underlined text
  • <s>: Strikethrough text
  • <sup>: Superscript text
  • <sub>: Subscript text
  • <mark>: Marks text with a highlight
What are input tag attributes?
    The <input> tag has many attributes that can be used to customize its behavior and appearance. Some common attributes include:
  • type: Specifies the type of input field (e.g., text, password, checkbox, radio, submit, button).
    name: Specifies the name of the input field, which is used to identify the field when the form is submitted.
    value: Specifies the default value of the input field.
    placeholder: Provides a hint to the user about the expected input.
    required: Indicates that the field must be filled in before the form can be submitted.
    disabled: Disables the input field.
    readonly: Makes the input field read-only, preventing the user from modifying its value.
What are form attributes?
    The <form> tag has several attributes that can be used to specify the form's behavior and appearance. Some common attributes include:
  • action: Specifies the URL of the page that will handle the form data when it is submitted.
  • method: Specifies the HTTP method used to submit the form data (e.g., GET, POST).
  • target: Specifies the target window or frame where the action will be performed.
What is a marquee tag?
  • The <marquee> tag is a deprecated HTML tag that was used to create scrolling text effects. However, it is no longer recommended for use due to its limited functionality and potential for accessibility issues.
What are the marquee tag attributes?
    The <marquee> tag had several attributes, including:
  • direction: Specifies the direction of the scrolling (e.g., left, right, up, down).
  • loop: Specifies the number of times the text should scroll.
  • scrollamount: Specifies the speed of the scrolling.
  • width: Specifies the width of the marquee.
  • height: Specifies the height of the marquee.
What are multimedia tags?
    Multimedia tags are HTML tags used to embed multimedia content (such as images, audio, and video) into web pages. Some common multimedia tags include:
  • <img>: Used to embed images.
  • <audio>: Used to embed audio files.
  • <video>: Used to embed video files.
Can we use block elements in inline elements?
  • No, you cannot use block elements within inline elements. Block elements are designed to occupy the entire width of their container and start on a new line, while inline elements are designed to flow with the text and do not start on a new line.
What are the differences between absolute and relative paths?
  • Absolute paths: Absolute paths specify the complete path to a file or directory, starting from the root directory of the file system. For example, /path/to/file.html is an absolute path.
  • Relative paths: Relative paths specify the path to a file or directory relative to the current directory. For example, ../images/image.jpg is a relative path that refers to an image file located one directory up from the current directory.
What are summary and details tags?
  • The <summary> and <details> tags are HTML5 elements used to create collapsible content sections. The <summary> tag defines the summary or title of the section, while the <details> tag contains the main content. Clicking on the summary text expands or collapses the details section.
What tag is helpful to give reviews?
  • The <article> tag is often used to represent a self-contained piece of content, such as a blog post, article, or review. It can be a good choice for organizing reviews on a webpage.
What are fieldset and legend tags?
  • <fieldset>: This tag is used to group related elements together within a form. It can be used to visually organize form elements into sections.
  • <legend>: This tag is used to define a caption for a <fieldset> element. It typically appears above the grouped elements.
